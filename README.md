# Java Guess Game 🎯

A simple console-based Number Guessing Game developed in Java. The program generates a predefined secret number and asks the user to guess it. It demonstrates the use of user input, conditional statements, and the Scanner class in Java.

## Features
- Takes input using `Scanner`
- Uses `if-else` statements
- Compares user input with the secret number
- Displays success or failure messages
- Beginner-friendly Java project

## Technologies Used
- Java
- VS Code
- JDK

## Concepts Covered
- Java Basics
- `Scanner` Class
- Variables
- User Input
- Conditional Statements (`if-else`)
- Console-Based Programming

## Sample Output
Enter a number (1-10): 7
Congratulations! You guessed correctly.
